# Hangman Game

Classic Hangman game built with **HTML, CSS & JavaScript**. Guess the hidden word by clicking letters. Wrong guesses draw the hangman, correct ones reveal letters.
Includes categories, win/lose sounds, and a restart button. Fun and interactive!

## Features

- Random word selection from multiple categories  
- Clickable letters for guessing  
- Hangman drawing updates with wrong guesses  
- Win and lose sound effects  
- Restart button to play again  
- Responsive design

## How to Play

1. Click on the letters to guess the word.  
2. Correct letters appear in the placeholders.  
3. Wrong guesses draw parts of the hangman.  
4. The game ends when the word is guessed or the hangman is fully drawn.

## Technologies Used

- HTML5  
- CSS3 (Flexbox & Grid)  
- Vanilla JavaScript
